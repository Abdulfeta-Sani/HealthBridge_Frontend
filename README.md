HealthBridge – Connecting Customers to Pharmacies

HealthBridge is a web-based platform designed to help customers easily locate nearby pharmacies and check the availability of medicines. The system aims to improve access to pharmacy information by providing a simple and user-friendly interface where users can search for pharmacies and view available medications.

This project was developed as a final-year capstone project by a team of Software Engineering students using an Agile development approach.

Project Overview
Finding a pharmacy that has a specific medicine can often take time and effort. HealthBridge addresses this problem by providing a centralized platform where users can:
  - Search for nearby pharmacies
  - View available medicines
  - Access pharmacy information in one place
The goal of the system is to make it easier for customers to find pharmacies and reduce the time spent searching for essential medications.

System Architecture
The system follows a standard full-stack architecture:
  - Frontend (React) → Backend API (Node.js + Express) → Database (MySQL)

The React frontend communicates with the backend through RESTful APIs to retrieve and display pharmacy and medicine data.
